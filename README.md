# Project: Omni-Sentient (OS-V1)
### Quantitative Surveillance & Execution Engine

Omni-Sentient is a private algorithmic trading framework designed for high-frequency market surveillance and risk-managed execution via FIX 4.4 and Open API protocols.

#### Core Architecture:
- **Surveillance Engine:** Multi-pair correlation monitoring using Information Entropy.
- **State Estimation:** Implementation of Kalman Filters for price-action noise reduction.
- **Veto-Gate Logic:** A proprietary 9-tier safety protocol for order validation.
- **Infrastructure:** Low-latency TCP socket bridges optimized for mobile-to-London (LD5) transit.

*Note: The core logic and execution modules of this project are kept in a separate, private environment for security and intellectual property protection.*
